Color Scheme Generator

for your designs

The Color Scheme Generator is a web application that allows users to effortlessly generate or browse color schemes for their designs. Developed using HTML, CSS, and JavaScript, this project leverages the Color API to generate color schemes based on user inputs.

The application offers a simple and intuitive interface that includes a color palette picker, a scheme picker, and a submit button to generate color schemes in real-time.
